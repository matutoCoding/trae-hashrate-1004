## 1. 产品概述

古籍修复补纸配纸生产力系统是面向古籍修复室的专业离线应用，旨在通过数字化手段提升古籍修复的科学性、精准度与工作效率。系统通过采集待修书页的多项物理指标，智能匹配最优补纸，辅助破损检测与修复方案制定，建立完整修复档案，实现纸库科学化管理。

- **目标用户**：古籍修复师、文献保护人员、博物馆/图书馆修复科室
- **核心价值**：提升配纸精准度、避免二次损伤、建立可追溯修复档案、规范化纸库管理
- **运行模式**：纯前端离线应用，所有数据本地存储，保护文物数据安全

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 修复师 | 本地创建 | 书页采集、配纸检索、破损标注、修复档案管理、纸库查询 |
| 管理员 | 本地创建 | 全部功能权限 + 纸库库存管理 + 系统配置 |

### 2.2 功能模块

1. **书页采集页**：待修书页基础信息与物理指标采集（帘纹间距、纤维成分、厚度、色度、酸碱度）
2. **配纸检索页**：多维度相似度匹配算法、补纸推荐排序、匹配度可视化展示
3. **破损标注页**：破损类型识别（虫蛀/霉斑/絮化/撕裂）、区域标注、搭口宽度计算、下捻方向建议、酸碱度相容性校验
4. **修复档案页**：修复前后影像管理、配纸批次记录、托裱平整度模拟、翘曲预测、塌陷风险预警
5. **纸库管理页**：按朝代/纸种分类管理、库存台账、补纸详细参数维护、配补用纸量清单生成

## 3. 核心流程

```mermaid
flowchart LR
    A["接收待修古籍"] --> B["书页信息采集\n（帘纹/纤维/厚度/色度/pH）"]
    B --> C["破损区域标注\n（类型识别+圈选）"]
    C --> D["配纸智能检索\n（多维度相似度匹配）"]
    D --> E["相容性校验\n（酸碱度+风险评估）"]
    E --> F["修复方案计算\n（搭口+下捻+尺寸）"]
    F --> G["托裱模拟预测\n（平整度+翘曲+塌陷）"]
    G --> H["执行修复操作"]
    H --> I["修复档案记录\n（影像+批次+用量）"]
    I --> J["纸库库存更新"]
```

**主要用户流程说明**：
1. 修复师接收待修古籍，首先在书页采集页录入书目信息并采集各项物理指标
2. 在破损标注页识别并圈选破损区域，系统自动计算破损面积和修复参数
3. 根据采集的指标在配纸检索页进行智能匹配，获取推荐补纸列表
4. 系统自动校验补纸与原纸的酸碱度相容性，避免二次损伤
5. 系统计算补纸搭口宽度、下捻方向和裁切尺寸，生成修复方案
6. 进行托裱模拟，预测平整度、翘曲风险和塌陷风险
7. 完成修复后，在修复档案页记录前后影像和配纸使用情况
8. 纸库库存自动更新，并可生成配补用纸量清单

## 4. 用户界面设计

### 4.1 设计风格

**设计理念**：古籍典雅与现代科技的融合。采用宣纸质感的米黄色调为基底，配合深褐墨色文字，点缀朱红印章色作为强调色，营造专业、庄重、富有文化底蕴的视觉体验。

- **主色调**：宣纸米白 (#F5F0E6)、深褐墨色 (#2C1810)
- **点缀色**：印章朱红 (#C83C23)、古籍石青 (#2E4A62)、竹简竹青 (#6B8E23)
- **警示色**：朽木黄 (#D4A017)、霉斑灰 (#6B6B6B)
- **按钮风格**：圆角矩形 (8px)、细边框、悬停微阴影、按压下沉效果
- **字体**：标题用"思源宋体"体现古籍韵味，正文用"思源黑体"保证可读性，数字用等宽字体
- **布局风格**：左右分栏主布局 + 卡片式内容区 + 古籍式留白边距
- **装饰元素**：细双线边框、米字格底纹、印章角标、纸质纹理背景
- **图标风格**：线性图标，2px描边，融入文房四宝、古籍、印章等元素

### 4.3 响应式设计

- **设计原则**：Desktop-first，兼顾平板横屏使用
- **断点设置**：1440px（标准）、1024px（平板横屏）、768px（平板竖屏）
- **主侧边栏**：≤1024px时转为可收起抽屉式菜单
- **数据表格**：≤768px时转为卡片式列表展示
- **画布操作**：触摸设备支持双指缩放和单指拖拽

### 4.4 微交互动效

- 页面切换：淡入淡出 + 轻微位移，300ms缓动
- 卡片悬停：阴影加深 + 轻微上浮，200ms过渡
- 按钮交互：悬停背景色变化，按压缩小96%
- 数据加载：骨架屏脉冲动画
- 标注操作：锚点呼吸灯效果
- 风险预警：边框闪烁 + 图标抖动
